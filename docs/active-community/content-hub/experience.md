# ⭐Начисление опыта _(XP)._    

Система опыта _(XP)_ в Контент-Хабе работает автоматически и начисляет опыт на основе вашей активности.

| Действие | Количество |
|----------|---------------|
| Публикация любого поста | +10 опыта |
| Обновление [`#Апдейт`] по ранее опубликованному проекту | +20 опыта |
| Уникальный [`#Уник`], то есть написанный вами, пост/материал | +30 опыта |
| Уникальное [`#Уник`] обновление [`#Апдейт`] | +40 опыта |
| Уникальный [`#Уник`] гайд [`#Гайд`] в формате Teletype, Telegraph и прочего | +60 опыта |

::: warning #Уник получает поинты только в случае, когда пост содержит более 300 символов.
:::

::: danger Полная проверка Контент-Хаба происходит раз в неделю в воскресенье/понедельник.
:::
