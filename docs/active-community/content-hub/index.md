# Гайд по Контент-Хабу♻️

## Быстрый старт в Контент-Хабе

### Введение в Контент-Хаб.

**Контент-Хаб** — Это место для обмена любого рода информацией между **ВСЕМИ** участниками Фармилки.

>Контент-Хаб является некой «БАЗОЙ данных», местом, куда стекает любая информация: копипаст, гемные проекты, ранние проекты, щиткоины, мусорные проекты, трейдинг сетапы и прочее. Информация из Контент-Хаба может использоваться для дальнейшей фильтрации, публикации в иных инструментах.

::: warning Ваша активность в Контент-Хабе отслеживается. За неё вам начисляется опыт, являющийся частью системы ролей с привилегиями.
:::
***

::: warning Доступ к Контент-Хабу предоставляется вместе с добавлением основной папки с инструментами из бота [@FarmersDaoBot](https://t.me/FarmersDaoBot): `Личный кабинет → Доступ → Папка с инструментами`
:::

***
### Основные правила.

1️⃣В Контент-Хаб можно постить что угодно, но делать это нужно на основе базовых правил и хэштегов.

2️⃣Участники Контент-Хаба получают опыт за свою активность. 

3️⃣Опыт влияет на получение ролей. Роли таят в себе привилегии.

4️⃣**Scout** — Самая распространённая и лёгкая в получении роль, открывающая полный доступ к Фармилке.

5️⃣Любой пост должен содержать как минимум **ОДИН** хэштег. Далее, если пост того требует, вы обязаны добавлять другие хэштеги.

6️⃣Если вы публикуете пост с обновлением по проекту, который освещался ранее, вы должны публиковать его в ответ на крайний пост по этому проекту  .

7️⃣За несоблюдение правил и неверный постинг вам начисляются штрафы, снижающие количество вашего опыта.

8️⃣Полная проверка Контент-Хаба происходит раз в неделю в Воскресенье/Понедельник.

9️⃣Полностью изученный гайд по Контент-Хабу — Залог качественной работы и меньшего количества штрафов.
