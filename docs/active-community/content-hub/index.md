# ♻️Гайд по Контент-Хабу.

## Введение в Контент-Хаб.

Итак, как Вы уже знаете из [основного гайда по Фармилке](/working-tools/content-hub), Контент-Хаб — это место для обмена любого рода информацией между **ВСЕМИ** участниками Фармилки, который служит некой «БАЗОЙ данных», местом, куда стекает любая информация: копипаст, гемные проекты, ранние проекты, щиткоины, мусорные проекты, трейдинг сетапы и прочее. 

::: warning Доступ к Контент-Хабу предоставляется вместе с добавлением основной папки с инструментами из бота [@FarmersDaoBot](https://t.me/FarmersDaoBot): `Личный кабинет → Доступ → Папка с инструментами`
:::

## Что важно понимать!

* Вся активность в Контент-Хабе отслеживается. За неё начисляется опыт на основе которого мы назначаем, повышаем роли и/или распределяем аирдропы.

* Информация из Контент-Хаба может использоваться участниками команды и участниками комьюнити для фильтрации и публикации в других инструментах. 

>Пример: Вы нашли в какой-то миррорке ресёрч интересного проекта, опубликовали его в Контент-Хаб, чтобы не потерять. Позже Вы вернулись к этому посту, отработали проект на нескольких аккаунтах и написали об этом в «Что Воркает Фармилка [To Do]», приложив ссылку на ресёрч из Контент-Хаба, чтобы не писать всё заново. 

## Основные правила.

Перед началом работы необходимо усвоить следующие правила:

1️⃣В Контент-Хаб можно постить что угодно, но делать это нужно на основе правил и хэштегов.

2️⃣Любой пост должен содержать как минимум **ОДИН** хэштег. Далее, если пост того требует, Вы обязаны добавлять другие хэштеги.

3️⃣Если Вы публикуете пост с обновлением по проекту, который освещался ранее, Вы должны публиковать его в ответ на крайний пост по этому проекту.

4️⃣За несоблюдение правил и неверный постинг Вам начисляются штрафы, снижающие количество вашего опыта.

5️⃣Полная проверка Контент-Хаба происходит раз в неделю в Воскресенье/Понедельник.

6️⃣Полностью изученный гайд по Контент-Хабу — Залог качественной работы и меньшего количества штрафов.
