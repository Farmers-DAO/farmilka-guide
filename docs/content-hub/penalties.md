# ⚠️Система штрафов

За несоответствие правилам публикаций, систематические нарушения с вас будут сниматься очки опыта.

| Нарушение | Штраф | 
|-----------|-------|
| Повторный пост | -30 опыта | 
| Неверный хэштег или оформление поста | -20 опыта | 
| Бессмысленные сообщения, флуд | -10 опыта | 
| Использование прямой функции телеграм «ПЕРЕСЛАТЬ» | -100 опыта | 
| Копипаст со ссылками на сторонние ресурсы | -100 опыта | 
| Упоминания или ссылки на сторонние ресурсы в статьях Teletype и других | -100 опыта |

::: warning Если вы планируете копипастить, вам необходимо полностью скопировать информацию, пост, статью, предварительно удалить все упоминания сторонних каналов, источников!
:::

::: danger Полная проверка Контент-Хаба происходит раз в неделю в Воскресенье/Понедельник.
:::
