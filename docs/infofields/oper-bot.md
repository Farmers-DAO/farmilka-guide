# Оперативный Бот

Оперативный Бот — это специальный инструмент для отслеживания новостей крипто проектов, позволяющий нам **ПЕРВЫМИ** получать официальные новости от проектов.

## Как вам работать с Оперативным Ботом?

> Оперативный Бот является особым инструментом нашего инфополя, это не парсер, не агрегатор и не слив сторонних приваток. 

Оперативный Бот автоматически отслеживает Discord и X новости тех проектов, которые указывает команда Фармилки, и присылает непосредственно нам в самого бота Telegram. 

::: warning Перейти в Оперативного Бота можно через основного бота Фармилки ([@FarmersDaoBot](https://t.me/FarmersDaoBot)) или по прямой ссылке: [@OP_mirrorBot](https://t.me/OP_mirrorBot)
:::

## Функционал бота.

### Получение новостей.
После запуска бота вы начнёте автоматически получать уведомления о новостях всех проектов, которые команда подключила к нему. Каждое сообщение поступает с переводом, а также содержит кнопку **«Оригинал»**, чтобы прочитать исходный текст.

### Быстрый доступ к источникам.
Под каждой новостью вы найдёте прямые ссылки на:

- **Первоисточник новости.**
- **Discord сервер проекта.**

Это позволит вам быстро перейти туда и узнать больше деталей.

### Поиск проектов.
В боте реализовано два способа поиска:

- **Быстрый поиск** — для оперативного нахождения проекта.
- **Поиск по алфавиту** — для удобного просмотра списка проектов, название которых начинается с определенной буквы.

Это позволит вам за считанные секунды найти нужный проект, даже если в боте подключено много проектов.

### Индивидуальные настройки.

1. Фильтрация проектов.
На главной странице бота **вы можете выбрать**, какие проекты вас интересуют. Просто отключите уведомления от ненужных проектов, оставив только релевантные вам.

2. Отключение всех уведомлений.
Во вкладке **«Настройки»** есть опция **«Отключить уведомления»**, которая моментально отключает уведомления от всех проектов.

3. Продвинутый фильтр.
Продвинутая функция фильтрации позволяет выбирать конкретные разделы/ветки Discord серверов каждого проекта. Включить её можно во вкладке **«Настройки»**.

> Например, в Discord сервере проекта 0G Labs есть три раздела с новостями: _mini-events; node-announcements; announcements_. С помощью продвинутого фильтра вы сможете подписаться только на интересующие вас разделы (_Например, только на «node-announcements»)_, чтобы не получать лишние сообщения.
