# Роль Minion.

Самая базовая роль, которая присваивается каждому участнику Фармилки.

## Как получить роль?
Роль Minion вы получаете как только покупаете Фармилку и вступаете в [Контент-Хаб](/working-tools/content-hub/).

::: warning Доступ к Контент-Хабу предоставляется вместе с добавлением основной папки с инструментами из бота [@FarmersDaoBot](https://t.me/FarmersDaoBot): `Личный кабинет → Доступ → Папка с инструментами`
:::

## Как сохранить роль? 

Минимальные требования для сохранения роли Minion — Отсутствуют.

## Привилегии роли.
* Роль Minion открывает вам **ДОСТУП К РЕФЕРАЛЬНОЙ ПРОГРАММЕ** Фармилки. Вы получаете 20% отчисления с каждой оплаты, приведённого вами реферала.
* Открывает возможность участия в аирдропе за [**полезную активность**](/roles-airdrops/valuable-activity-rewards/).
* Открывает **ДОСТУП К ЭКСКЛЮЗИВНЫМ** материалам _(Обучения, схемы, сливы)_ **ПЕРВОГО** уровня.
