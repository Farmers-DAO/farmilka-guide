# fNodes.

Инструмент, основной задачей которого служит заработок на нодах.

## Зачем Вам fNodes?

- Позволяет Вам выбирать наиболее актуальные проекты для запуска нод.
- Позволяет Вам своевременно получать обновления по нодам.
- Позволяет Вам изучать обучающий/базовый материал по работе с нодами.
- Позволяет Вам получать подсвеченный контент из инфополя Фармилки, связанный с нодами.

## Как Вам работать с fNodes?

Среди основных веток fNodes:

- «[F.A.Q [Как всё устроено]](https://t.me/c/2475391574/848)» — Объясняет Вам принцип работы инструмента _(информация совпадает с гайдом_).
- «[Обучающий материал [БАЗА]](https://t.me/c/2475391574/14/)» — Содержит обучающий и прочий полезный материал, связанный с работой по нодам.
- «[Миррор / Подсвеченные мувы](https://t.me/c/2475391574/495)» — Избавляет Вас от необходимости самостоятельного поиска и анализа полезной и потенциально-прибыльной информации в инфополе Фармилки. Ежедневно в этой ветке публикуется самый интересный контент по нодам из **ОСНОВНОГО** инфополя Фармилки. 
- «[Чат](https://t.me/c/2475391574/39)» — Чат, где Вы можете общаться по теме нод.

***

**Все другие ветки — опциональны** _(могут удаляться и создаваться)_. Они создаются для отслеживания обновлений по нодам конкретных проектов _(например, Aztec, Nexus)_. Удаляются они по мере окончания отработки.

>Они имеют схожий принцип работы с ветками в fRetrodrops, но гораздо проще: стикеров, эмодзи батареек в fNodes не существует.

### Существуют только хэштеги, сопровождающие любой пост.
**Всего существует 3 хэштега:**

- **#️⃣НазваниеПроекта** — Указывает на название проекта _(#Aztec)_. 
- **#️⃣Апдейт** — Указывает на то, что данный пост служит обновлением по проекту. Самые первые посты о проектах не имеют этого хэштега.
- **#️⃣Результат** — Указывает на завершение отработки и ставится в финальном посте о проекте.
