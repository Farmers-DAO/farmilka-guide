# fP2E.

Инструмент, основной задачей которого служит заработок на GameFi секторе, в частности на P2E играх.

## Зачем вам fP2E?

- Позволяет Вам выбирать наиболее актуальные P2E проекты.
- Позволяет Вам своевременно получать обновления по проектам.
- Позволяет Вам получать подсвеченный контент из инфополя Фармилки, связанный с P2E проектами.

## Как вам работать с fP2E?

- «[F.A.Q [Как всё устроено]](https://t.me/c/1991862193/1955)» — Объясняет Вам принцип работы инструмента _(информация совпадает с гайдом)_.
- «[Миррор / Подсвеченные мувы](https://t.me/c/1991862193/1611)» — Избавляет Вас от необходимости самостоятельного поиска и анализа полезной и потенциально-прибыльной информации в инфополе Фармилки. Ежедневно в этой ветке публикуется самый интересный контент по P2E играм из **ОСНОВНОГО** инфополя Фармилки. 
- «[Чат](https://t.me/c/1991862193/1966/)» — Чат, где Вы можете общаться по теме P2E игр.

***

**Все другие ветки — опциональны** _(могут удаляться и создаваться)_. Они создаются для отслеживания обновлений по P2E играм _(например, Cambria)_. Удаляются они по мере окончания отработки.

>Принцип работы в точности схож с fNodes.

### Существуют только хэштеги, сопровождающие любой пост.
**Всего существует 3 хэштега:**

- **#️⃣НазваниеПроекта** — Указывает на название проекта (_#Cambria_). 
- **#️⃣Апдейт** — Указывает на то, что данный пост служит обновлением по проекту. Самые первые посты о проектах не имеют этого хэштега.
- **#️⃣Результат** — Указывает на завершение отработки и ставится в финальном посте о проекте.
