# fTONMoves

Инструмент, созданный для заработка в экосистеме TON/Telegram.

## Зачем вам fTONMoves?

- Позволяет Вам отслеживать деятельность, активность и коллы участников команды, Тим-Воркеров, чья **ОСНОВНАЯ РАБОТА СТРОИТСЯ** на TON/Telegram экосистеме.
- Позволяет Вам изучать обучающий/базовый материал по работе с TON/Telegram экосистемой.
- Позволяет Вам получать подсвеченный контент из инфополя Фармилки, связанный с TON/Telegram.
- Позволяет Вам пользоваться софтами/скриптами для различных активностей из экосистемы TON/Telegram.
- Позволяет Вам изучать различные мошеннические/скам схемы из экосистемы TON/Telegram.
- Позволяет Вам пользоваться дополнительным инструментарием (_парсеры/агрегаторы_), созданным для TON/Telegram экосистемы.

## Как вам работать с fTONMoves?

В fTONMoves существует 6 базовых веток:

- «[F.A.Q [Как всё устроено]](https://t.me/c/2151249897/512)» — Объясняет вам принцип работы инструмента (_Информация совпадает с гайдом_).
- «[Обучающий материал [БАЗА]](https://t.me/c/2151249897/519/)» — Содержит обучающий и прочий полезный материал, связанный с работой по TON/Telegram экосистеме.
- «[Миррор / Подсвеченные мувы](https://t.me/c/2151249897/523)» — Ежедневно в этой ветке публикуется самый интересный контент по TON экосистеме из **ОСНОВНОГО** инфополя Фармилки. 
- «[Софт / Автоматизация](https://t.me/c/2151249897/524)» — Позволяет вам пользоваться различными софтами, скриптами, созданными для TON экосистемы, среди них: софт на покупку подарков и стикеров.
- «[SCAM / Схемы](https://t.me/c/2151249897/525/69569)» — Позволяет Вам отслеживать методы мошенничества из TON/Telegram экосистемы.
- «[Stickers Digest](https://t.me/c/2151249897/387065)» — Позволяет Вам пользоваться стикер-монитором, отображающим ежедневные тенденции на рынке стикеров.
- «[Чат](https://t.me/c/2243285093/5/)» — Чат, где вы можете общаться по теме TON экосистемы.

***

**Оставшиеся ветки — опциональны** (_Могут удаляться и создаваться_). Они служат личными ветками команды Фармилки, Тим-Воркеров. 

Названия веток совпадают с юзернемеймами/именами, а справа, в квадратных скобках, указана позиция этого человека:

- **[Team]** — Является участником команды.
- **[Solo Caller]** — Обладает ролью Тим-Воркера.
- **[TEST]** — Участник, проходящий стажировку на право дальнейшего ведения личной ветки в этом инструменте.

Внутри этих веток в свободном формате участники освещают свои сделки, коллы и всё, что связано с их работой по TON/Telegram экосистеме.

Вам эти ветки позволяют следить за конкретным человеком (_или несколькими_), демонстрирующим хорошие результаты.
