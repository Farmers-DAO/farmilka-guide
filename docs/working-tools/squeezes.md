# Выжимки.

Инструмент, в котором публикуются выжимки _(дайджесты)_ из наших инструментов.

## Зачем вам Выжимки?

- Позволяет Вам изучать регулярные сводки информации из наших инструментов.

## Как вам работать с инструментом Выжимки?

Достаточно просто вступить в канал с выжимками, где каждый пост — отдельный дайджест по определенному инструменту _(названия указываются в заголовке поста)._
