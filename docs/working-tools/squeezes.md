# Выжимки

Инструмент, в котором публикуются выжимки (_дайджесты_) из наших инструментов.

## Зачем вам Выжимки?

- Позволяет Вам изучать регулярные сводки информации из наших инструментов.

## Как вам работать с инструментом Выжимки?

Достаточно просто вступить в канал с выжимками, гд каждый пост — отдельный дайджест по определенному инструменту _(Названия указываются в заголовке поста)._
